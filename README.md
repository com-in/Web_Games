# Web_Games

一个包含多个Web游戏的项目，提供游戏导航和游戏入口。

## 项目结构

```
Web_Games/
├── docs/                 # 游戏导航网站
│   ├── index.html        # 游戏导航主页
│   └── Games/            # 游戏目录
│       └── Seven_Hours_At_Family_Web/  # 游戏文件夹
│           ├── logo/     # 游戏logo
│           └── play/     # 游戏入口
├── .gitattributes        # Git LFS配置
├── LICENSE              # 许可证文件
└── README.md            # 项目说明
```

## 如何使用

1. 克隆或下载本项目
2. 使用浏览器打开 `docs/index.html` 文件
3. 在游戏导航页面选择要游玩的游戏
4. 点击"立即游玩"按钮进入游戏
5. 点击"详情信息"按钮查看游戏详情

## 游戏列表

### Seven Hours At Family Web
- **版本**: 1.0.0
- **类型**: 冒险解谜
- **标签**: 恐怖、解谜、冒险
- **简介**: 一款恐怖冒险解谜游戏，玩家需要在7小时内解决各种谜题。

## 技术栈

- **前端**: HTML5, CSS3, JavaScript
- **样式**: 现代化渐变设计，毛玻璃效果
- **布局**: 响应式设计，适配不同屏幕尺寸
- **动画**: CSS过渡和变换效果
